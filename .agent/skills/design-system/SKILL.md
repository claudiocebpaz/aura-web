# Design System: Eximia Style Authority

## 1. Filosofía: "The Dark Tech Luxury"
Buscamos una estética que parezca un software de alta tecnología del año 2030. El diseño debe ser "limpio pero denso en detalles".

## 2. Tokens de Diseño (Tokens)
- **Fondo:** `#000000` (Negro absoluto). Proporciona profundidad infinita.
- **Superficies:** `#0A0A0A` (Gris casi negro) con un borde sutil `white/10`.
- **Acento:** `#3B82F6` (Azul Eléctrico) para elementos que necesitan "energía".
- **Radio:** `24px` (`rounded-3xl`) para tarjetas Bento. Proporciona un look moderno y amigable.

## 3. El Layout "Bento Grid"
Las secciones de servicios DEBEN ser asimétricas. 
- Una cuadrícula de 12 columnas donde las cajas varían en tamaño.
- **Glassmorphism:** Todas las cajas deben tener `backdrop-blur-xl` y un fondo semitransparente.

## 4. Tipografía
- **Headings:** 'Inter' o 'Space Grotesk'. Extra Bold, con espaciado entre letras negativo (`tracking-tighter`).
- **Body:** 'Inter' regular. Legibilidad máxima.
