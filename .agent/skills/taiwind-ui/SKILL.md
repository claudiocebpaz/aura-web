# Skill: Tailwind UI Specialist (Premium Implementation)

## 1. Anatomía del Componente
Cada componente debe ser "Production-Ready". Esto significa incluir:
- **Estados:** Hover, Focus-visible, Active y Disabled.
- **Accesibilidad:** Uso de ARIA roles y componentes de Headless UI para garantizar que sea navegable por teclado.
- **Refinamiento:** Un componente no está terminado sin un `ring-offset`, una sombra sutil o una transición de color.

## 2. Patrones de "Agencia de IA"
Implementar componentes específicos de marketing tecnológico:
- **Sticky Navbar:** Con efecto de desenfoque (Glassmorphism).
- **CTA Sections:** Con jerarquía visual clara (Botón primario brillante vs secundario fantasma).
- **Comparison Tables:** Estructuras limpias para contrastar servicios.

## 3. Dependencias Obligatorias
- **Headless UI:** Para lógica de UI compleja.
- **Heroicons:** Para una iconografía limpia y consistente.
